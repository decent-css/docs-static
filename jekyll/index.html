A Functional CSS Library

This is Functional CSS – f(scss).

Alternatively refered to as Utility Classes, the term Functional CSS came about as we found that we could use this method to develop entire websites and applications.

{% include section.html heading="Getting Started" body="Our goal with Decent is to provide a functional CSS library that makes no assumption about your design. It works best when built into your asset compliation, allowing you to utilize the settings and configure the classes to meet your exact needs." %}

Install Decent

$ npm install decent-scss

or if you're not using NPM in your project:

Download the Source

Import it into your Build

// Import settings before Decent modules.
@import 'settings';

// Then import Decent.
@import 'path/to/decent-scss/modules/all';

{% include section.html heading="Lengths" body="The lengths module provides fixed, percentage, min, and max widths and heights. The classes are configuragble by a SCSS variable list to allow for an infinite number of increments – though we recommend limiting the number of increments to about 10 based off a modular scale." %}

Markup

<div class="relative">
    <div class="w-1">  <!-- 0.8rem --> </div>
    <div class="w-2">  <!-- 1.6rem --> </div>
    <div class="w-3">  <!-- 2.2rem --> </div>
    <div class="w-4">  <!-- 2.6rem --> </div>
    <div class="w-5">  <!-- 4.2rem --> </div>
    <div class="w-6">  <!-- 6.8rem --> </div>
    <div class="w-7">  <!--  11rem --> </div>
    <div class="w-8">  <!--  18rem --> </div>
    <div class="w-9">  <!--  29rem --> </div>
    <div class="w-10"> <!--  46rem --> </div>
    <div class="w-11"> <!--  75rem --> </div>
  </div>

Settings

$length-increments: 0.8 1.6 2.2 2.6 4.2 6.8 11 18 29 46 75;
